# DevOps Monitoring API

A production-ready REST API for system metrics monitoring, 
built with Python Flask and containerized with Docker.

## Endpoints

| Endpoint | Description |
|----------|-------------|
| GET /health | Application health check |
| GET /metrics | CPU, RAM, and disk metrics |
| GET /info | Server information |

## Run Locally

docker-compose up --build
